# 小红书 发布指南

## 适用 providerIdentifier
- `xiaohongshu`

## 发布流程

### 图文笔记（主要）
1. 打开 `https://creator.xiaohongshu.com/publish/publish`
2. 点击"上传图片"区域，上传图片（最多 18 张）
3. 等待所有图片上传完成
4. 填写标题（从 `settings.title` 或 `content` 第一行提取）
5. 填写正文（`content`，纯文本）
6. 添加话题标签：点击 # → 输入话题名 → 选择匹配项
7. 选择地点（如 `settings` 中指定）
8. 点击"发布笔记"
9. 等待发布成功
10. 复制笔记 URL 作为 `releaseURL`

### 视频笔记
1. 打开 `https://creator.xiaohongshu.com/publish/publish`
2. 切换到"视频"标签
3. 上传视频，等待处理完成
4. 设置封面
5. 填写标题和正文
6. 添加话题标签
7. 发布

## 注意事项
- 图片限制：最多 18 张，建议 3:4 竖版比例
- 标题限制：20 字符
- 正文限制：1000 字符
- 话题标签：建议 3-5 个，选择推荐话题优先
- 小红书创作者中心需要登录，确保 Chrome Profile 保持登录状态
- 发布后可能有审核期（几分钟到几小时），发布成功不代表立即可见
